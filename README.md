# Escola de Música DeCifrei

A criação desse site foi inspirado na escola de música do meu ex-professor e amigo, Caique.
A tentativa foi divulgar as aulas e os professores que fazem parte.

### Sobre o Site

**1. Home:** Visão geral do site, mostrando os cursos oferecidos, professores, canal no youtube, etc.

**2. Cursos:** Mostra cursos já gravados disponíveis em outras plataformas, como Hotmart.

**3. Aulas Presenciais:** Abre uma aba de inscrição para fazer aulas presenciais

**4. Sobre nós:** Redireciona para a parte de professores da escola.

**5. Loja:** Se a idéia da empresa DeCifrei ir para frente, há de se tornar uma aba para venda de instrumentos e acessórios músicais
